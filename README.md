## 🚩 Goal

🔖 **깨끗하게!** 클린 코드! 다른 팀원도 잘 알아볼 수 있도록 가독성 좋은 코드로 작성하기

🔖 **즐겁게!** 시작부터 끝까지 불화 없이 서로 돕는 팀 분위기

🔖 **완성도 있게!** 양보다 질! 필요한 기능과 불필요한 기능을 판별 → 분량은 최적화, 완성도 있는 결과물 만들기

 주요기능
  - 회원가입/로그인
  - 지역별, 음식 카테고리별 필터링

  - 식당 예약
  - 리뷰 쓰기
  - 내 예약, 주문 확인
  - 관심 장소 저장
